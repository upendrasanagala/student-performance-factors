# Student Performance Factor Analysis
Overview

This project analyzes different factors that influence student academic performance using data analysis and machine learning techniques. The goal is to identify which factors have the greatest impact on student exam scores and overall academic outcomes.

The analysis considers multiple variables such as attendance, parental support, internet access, extracurricular activities, sleep duration, motivation level, tutoring support, family income, peer influence, and exam scores. 

student performance factors

# Problem Statement

Student performance is influenced by several academic, social, and lifestyle factors. Understanding these factors can help educators and policymakers design better strategies to improve learning outcomes.

Key Insights

Attendance and motivation strongly influence student performance.

Parental involvement has a positive correlation with exam results.

Adequate sleep and consistent study habits improve academic outcomes.
